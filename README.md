
# 📚 ProyectoLibreria

¡Bienvenido a **ProyectoLibreria**! Este proyecto es una aplicación de consola desarrollada en Java que permite gestionar una colección de libros. Proporciona funcionalidades para agregar, buscar, eliminar y listar libros, facilitando así la administración de una biblioteca personal o académica.

## 🛠️ Tecnologías Utilizadas

- ☕ **Java**: Lenguaje de programación principal.
- 🧰 **JDK 17**: Kit de desarrollo de Java.
- 🖥️ **IDE**: Compatible con IntelliJ IDEA, Eclipse o NetBeans.
- 📦 **Maven**: Para la gestión de dependencias (opcional).

## 📁 Estructura del Proyecto

```
ProyectoLibreria/
├── src/
│   └── Milibreria.java
├── README.md
└── LICENSE
```

- `Milibreria.java`: Clase principal que contiene la lógica de la aplicación.
- `README.md`: Documentación del proyecto.
- `LICENSE`: Información sobre la licencia del proyecto.

## 🚀 Cómo Ejecutar

1. **Clonar el repositorio**:

   ```bash
   git clone https://github.com/Zeqii/ProyectoLibreria.git
   cd ProyectoLibreria/src
   ```

2. **Compilar el proyecto**:

   ```bash
   javac Milibreria.java
   ```

3. **Ejecutar la aplicación**:

   ```bash
   java Milibreria
   ```

   > 💡 Asegúrate de tener Java instalado y configurado en tu sistema.

## 📚 Funcionalidades

- ➕ **Agregar Libro**: Permite añadir un nuevo libro proporcionando título, autor y año de publicación.
- 🔍 **Buscar Libro**: Busca libros por título o autor.
- 🗑️ **Eliminar Libro**: Elimina un libro existente de la colección.
- 📄 **Listar Libros**: Muestra todos los libros registrados en la biblioteca.
- ❌ **Salir**: Cierra la aplicación de forma segura.
